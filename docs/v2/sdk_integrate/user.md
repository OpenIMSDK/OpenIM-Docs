# 用户相关接口

包含查询用户资料、修改个人资料以及把某人加入黑名单（从黑名单移除）。注意，黑名单与好友无关，对于好友、非好友都可以拉黑。

| SDK               | 描述                   |
| :---------------- | :--------------------- |
| [GetUsersInfo]    | 获取用户资料           |
| [SetSelfInfo]     | 登录用户修改个人资料   |
| [GetSelfUserInfo] | 登录用户获取个人资料   |
| [AddBlack]        | 把某人拉黑             |
| [GetBlackList]    | 添加某用户到黑名单中   |
| [RemoveBlack]     | 把某用户从黑名单中移除 |
